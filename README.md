# Housekeeping Services Website

A professional housekeeping services website built with HTML, CSS, and JavaScript.

## Project Structure

```
housekeeping-services/
│
├── index.html
├── about.html
├── services.html
├── contract-request.html
├── careers.html
├── contact.html
│
├── assets/
│   ├── css/
│   │   ├── style.css              # Main styles
│   │   ├── responsive.css         # Responsive design styles
│   │   └── animations.css         # Animation effects
│   │
│   ├── js/
│   │   ├── main.js                # Main JavaScript functionality
│   │   ├── contract-form.js       # Contract request form logic
│   │   └── careers-form.js        # Careers application form logic
│   │
│   ├── images/
│   │   ├── logo.png               # Company logo
│   │   ├── hero-banner.jpg        # Homepage banner
│   │   ├── services/              # Service-related images
│   │   └── team/                  # Team member photos
│   │
│   └── icons/                     # Icon assets
│
├── privacy-policy.html
├── terms.html
│
└── README.md
```

## Pages

- **index.html** - Homepage with hero banner and featured services
- **about.html** - About the company and team information
- **services.html** - Detailed service offerings
- **contract-request.html** - Contract request form
- **careers.html** - Career opportunities and application form
- **contact.html** - Contact information and contact form
- **privacy-policy.html** - Privacy policy document
- **terms.html** - Terms and conditions

## Features

- Responsive design for mobile, tablet, and desktop
- Smooth animations and transitions
- Professional form handling for contracts and career applications
- Clean and organized file structure

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through the different pages using the navigation menu
3. Fill out forms for contract requests or career applications

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Notes

- Ensure all image assets are placed in the `assets/images/` directory with appropriate subdirectories
- Font and icon files should be placed in the `assets/icons/` directory
- Customize the CSS files to match your branding
